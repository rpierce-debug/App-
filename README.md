# Cerina

Cerina is a lightweight mental health companion you can run as a static page. It offers a daily check-in, coping skill reminders, and a local-only reflection log that keeps the latest 50 entries in your browser.

## Features
- Guided daily check-in with mood, energy, and notes plus live mood labels and character counter
- Optional breathing break reminder using box breathing and quick prompt suggestions
- Local journal that summarizes your last check-in, weekly average mood, streaks, and shows a 10-entry mood trend line
- Filterable timeline (all, 7d, 30d) with export to JSON and clear actions
- Crisis resources and grounding exercise references

## Running locally
Open `index.html` in your browser. All data is stored in `localStorage` on your device.
